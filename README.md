# MK Portfolio

Professional, stylish, animated portfolio built with Next.js, TypeScript, Tailwind, and Framer Motion.

## Tech
- Next.js (App Router) + TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Development
1. Install dependencies
2. Run the dev server

## Deployment (Netlify)
- This repo includes `netlify.toml` and uses the official Next.js Netlify plugin.
